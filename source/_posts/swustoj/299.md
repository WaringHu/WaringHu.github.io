# 题目

> 用递归的方法求 f(n) = 1 * 1 + 2 * 2 + 3 * 3 +……+ n * n

<!-- more -->

## 输入

> 输入数字n

## 输出

> 输出结果 f(n)

## 样例输入

> 1
3

## 样例输出

> 1
14

# 源代码

```cpp
#include <stdio.h>
int f(int n)
{
	if (n == 1)
	{
  		return 1;
	}
	return f(n-1)+n*n;
}

int main()
{
	int n;
	while (scanf("%d", &n) == 1)
	{
 		printf("%d\n", f(n));
	}
	return 0;
}
```
