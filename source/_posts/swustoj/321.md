# 题目

> 阶乘(factorial)是基斯顿·卡曼(Christian Kramp, 1760 – 1826)于1808年发明的运算符号。 任何大于1的自然数n阶乘可以表示为以下形式： n!=n×(n-1)! 另外，数学家定义，0！=1，所以0！=1！ 你的任务是编写一个程序来计算一个给定数字n的阶乘n!（n<14）

<!-- more -->

## 输入

> 输入数据只有一行，即数字n的值，（n<14)。

## 输出

> 要求输出n!的值

## 样例输入

> 5

## 样例输出

> 120

# 源代码

```cpp
#include <stdio.h>

int main()
{
	int i, n;
	int sum = 1; 
    scanf("%d", &n);
    for(i=1; i<=n; i++)
    {
        sum *= i;
    }
    printf("%d\n", sum);
    return 0;
}
```
