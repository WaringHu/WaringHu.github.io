# 题目

建立一个长度为 n 的带头结点的双向链表，使得该链表中的数据元素递增有序排列。（必须使用双向链表完成，数据类型为整型。）

<!-- more -->

## 输入

> 第一行：双向表的长度； 
> 第二行：链表中的数据元素。

## 输出

> 输出双向链表中的数据元素的值。

## 样例输入

> 10
> 2 4 6 3 5 8 10 21 12 9

## 样例输出

> 2 3 4 5 6 8 9 10 12 21

# 源代码

```cpp
#include <iostream>
#include <list>
#include <algorithm>

using namespace std;

int main()
{
    list<int> List;
    int m;
    cin >> m;
    int arr[1000];
    for(int i=0; i<m; i++)
    {
        cin >> arr[i];
        List.push_back(arr[i]);
    }
    list<int>::iterator iter;
    List.sort();
    for(iter = List.begin(); iter != List.end(); iter++)
    {
        cout << *iter << " ";
    }
    return 0;
}
```
