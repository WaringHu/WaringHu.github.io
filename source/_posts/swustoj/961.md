# 题目

建立顺序栈或链栈，编写程序实现十进制数到二进制数的转换。

<!-- more -->

## 输入

> 输入只有一行，就是十进制整数。

## 输出

> 转换后的二进制数。

## 样例输入

> 10

## 样例输出

> 1010

# 源代码

```cpp
#include <iostream>
#include <list>
#include <algorithm>

using namespace std;

int main()
{
    list<int> List;
    int m;
    cin >> m;
    while(m)
    {
        List.push_front(m%2);
        m = m/2;
    }
    list<int>::iterator iter;
    for(iter = List.begin(); iter != List.end(); iter++)
    {
        cout << *iter;
    }
    return 0;
}
```
